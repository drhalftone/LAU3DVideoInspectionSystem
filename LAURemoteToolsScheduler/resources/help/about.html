A configuration tool for automated inspection system video recording systems.

What This Tool Does:

  • Configures daily recording schedules for system monitoring
  • Sets up cloud storage paths for video archives
  • Manages RFID object ID encoding integration
  • Configures LAUOnTrakWidget for automatic startup (relay control)
  • Creates Windows Task Scheduler entries for automation

Recording Workflow:

  1. Startup: LAUOnTrakWidget launches on login for camera power cycling
  2. Record: LAU3DVideoRecorder captures video at scheduled time
  3. Encode: LAUEncodeobjectIDFilter extracts RFID tags and embeds object IDs
  4. Upload: Videos are copied to cloud storage (OneDrive/SharePoint)
  5. Cleanup: Local temporary files are removed

Hardware Setup:

Power Supply → OnTrak USB Relay → PoE Injector → Cameras
LAUOnTrakWidget provides hardware-level power cycling when cameras freeze.

Configuration File:

Settings are saved to systemConfig.ini in the application directory.
The generic recordVideo.cmd batch script reads this file to execute the workflow.
