JETR Calibration Manager

Version: 1.0.0

A Qt-based application for viewing, editing, and managing JETR (Just-Enough-To-Reconstruct) vectors used in 3D reconstruction from RGB-D camera data, with specialized support for historical 3D video analysis.

Key Features:

  • History Management: Load and manage camera configuration changes over time
  • Visual Progress: Track calibration completion with date-based checkboxes
  • Multi-Camera Support: Handle stacked camera configurations with individual calibration
  • Smart Storage: Automatic persistence of calibration work between sessions
  • Incremental Workflow: Work on a few dates at a time with saved progress
  • Python Integration: Generate history files with embedded script support

JETR Vectors:

37-element arrays containing minimal parameters for 3D reconstruction:

  • Intrinsic Parameters (0-11): Camera calibration and distortion
  • Extrinsic Parameters (12-27): Camera pose in world coordinates
  • Bounding Box (28-33): 3D region of interest
  • Depth Processing (34-36): Scale factors and depth limits

Architecture:

Main Window: History file management, date navigation, progress tracking

JETR Editor: Visual camera identification, parameter editing, inventory management

Part of the LAU3DVideoRecorder project for 3D video inspection systems.
