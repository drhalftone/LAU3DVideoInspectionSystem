LAU OnTrak Widget

Version 1.0

A professional control interface for OnTrak USB relay devices used in equipment automation.

Key Features:

  • Relay Control: Manual toggle control for relays K0 and K1
  • Safety Protection: Daily cycle limit (3 cycles/day) to prevent equipment damage
  • Remote Control: IPC server enables control from external applications
  • Visual Feedback: Color-coded status indicators for relay states
  • Demo Mode: Full functionality simulation when no device is connected

Remote Control via IPC:

External applications can connect to this widget using inter-process communication:

  • CYCLE_RELAYS - Automated relay cycling for equipment operation
  • STATUS - Query current operational state
  • GET_LIMIT - Check remaining daily cycles

Safety & Reset Information:

Why Daily Limits Exist: Equipment protection against excessive mechanical stress from relay cycling.

When Remote Reset is Needed:

  • After equipment maintenance or calibration
  • When authorized for extended testing procedures
  • During emergency operational requirements
  • When the daily limit was reached due to automated system operations

Status Colors:

  • Red: Relay OFF
  • Green: Relay ON
  • Blue: Processing/waiting state
  • Yellow: Demo mode (no device connected)
