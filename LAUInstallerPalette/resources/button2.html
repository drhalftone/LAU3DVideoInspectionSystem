HOW IT WORKS:
• When you click: Shows Task Scheduler setup instructions
• What to configure: Create Windows Task Scheduler entry to launch LAUOnTrakWidget.exe at login
• Status check: button2_test.ps1 checks if LAUOnTrakWidget.exe process is running
• Button color: GREEN if process running, RED otherwise

WHY THIS IS NEEDED:
The OnTrak widget controls power to the cameras via USB relay. Cameras must be powered on before they can be detected and used. The Task Scheduler ensures OnTrak launches automatically at system startup.

WHAT THE ONTRAK WIDGET DOES:
• Detects OnTrak USB relay hardware
• Controls camera power on/off via relay
• Allows cameras to warm up before recording sessions
• Runs in system tray for easy access

PREREQUISITES:
• Button 1 GREEN: systemConfig.ini must exist
• OnTrak USB relay hardware connected to the system
