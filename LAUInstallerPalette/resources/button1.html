SYSTEM CONFIGURATION

Configure system settings and operational parameters for the 3D video inspection system.

STATUS: {{STATUS_ICON}} {{STATUS_TEXT}}

HOW IT WORKS:
• When you click: Launches LAURemoteToolsScheduler.exe
• What the scheduler does: Collects configuration data and saves it to systemConfig.ini
• Status check: button1_test.ps1 looks for systemConfig.ini
• Button color: GREEN if systemConfig.ini exists, RED otherwise

WHY THIS IS NEEDED:
The systemConfig.ini file contains essential settings (camera positions, storage paths, location identifiers, etc.) required by all other tools in the system. Without this configuration file, subsequent setup steps cannot proceed.

WHAT TO CONFIGURE:
The scheduler tool will prompt you to set:
• Location identifier and installation paths
• Local temporary storage directory
• Camera position labels
• Task scheduler settings for automated recording
• Power control settings (OnTrak integration)

PREREQUISITES:
None - this is the first setup step.
